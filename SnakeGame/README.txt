Controls:
 - Movement: Arrow Keys or WASD
 - Pause/Unpause Game: P
 - Throw Apple (stages two and four): SPACE

Custom Features:
 - Highscore System
 - Four Difficulty Stages
     - Traditional Snake
     - Simple Obstacles, Random
     - Moving Obstacles
           - Patterns, Random
     - Obstacle Blitz, Random
 - Throw Apples to Destroy Obstacles
 - Powerup System
     - Watermelon - Speed Up Game for 50 Moves
     - Ham - Slow Down Game for 50 Moves
     - Picnic Basket - 10 Apples, Reset Game Speed
 - Graphics
 - Sounds
 - Information Dashboard
